# Executive Summary

## Business problem

Modern organisations face ransomware, phishing, credential theft, insider risk, supply chain risk, misconfiguration, and data breach exposure. A single firewall, antivirus platform, or backup system is not enough. Security must be layered, monitored, tested, and governed.

## Proposed solution

This project proposes a defense-in-depth model that protects the organisation using multiple independent but coordinated controls. The architecture assumes that any individual control can fail, and therefore designs compensating controls around prevention, detection, response, and recovery.

## Target outcomes

- Reduce attack surface across network, endpoint, identity, cloud, and data layers
- Improve visibility through central logging and SIEM monitoring
- Strengthen resilience through tested backups and disaster recovery procedures
- Improve accountability using risk ownership, change control, and audit tracking
- Support business operations without compromising security

## Recruiter-facing value

This project demonstrates senior-level thinking expected from an AVP IT Networks and Infrastructure Support Manager, Network Security Engineer, Infrastructure Security Engineer, or Cybersecurity Engineer. It shows the ability to connect technology implementation with business risk reduction.
