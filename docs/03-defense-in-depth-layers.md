# Defense-in-Depth Layers

## 1. Governance layer

Controls:

- Security policies and standards
- Risk register and risk acceptance process
- Change approval board
- Audit issue tracking
- Supplier and third-party access reviews

Evidence of capability:

- Able to communicate security risk to leadership
- Able to prioritise remediation based on business impact
- Able to link technical controls to audit and compliance requirements

## 2. Perimeter security layer

Controls:

- Next-generation firewall policies
- Intrusion prevention
- VPN with MFA
- DMZ design
- Deny-by-default inbound access
- Rule recertification and clean-up

Example experience represented:

- Check Point firewall operations
- Cisco ASA / Firepower / FTD exposure
- FortiGate multi-vendor firewall environments
- Site-to-site VPNs and secure third-party connectivity

## 3. Network segmentation layer

Controls:

- VLAN and VRF separation
- ACLs between user, server, management, and third-party zones
- Firewall enforcement between critical zones
- Secure routing and high-availability design
- Separate management network

Example experience represented:

- Enterprise switching and routing
- Data centre network support
- BGP, OSPF, EIGRP, MPLS, HSRP, EtherChannel, QoS
- ISP peering and upstream connectivity experience

## 4. Identity and access layer

Controls:

- MFA for remote and privileged access
- Role-based access control
- Joiner/mover/leaver access process
- Privileged account review
- Conditional access principles

## 5. Endpoint layer

Controls:

- Endpoint detection and response
- Antivirus/anti-malware
- Patch management
- Device control
- Secure baseline builds
- Local admin restriction

Example experience represented:

- Trend Micro Apex One
- Microsoft Defender concepts
- ManageEngine Endpoint Central
- Endpoint hardening and patching

## 6. Email and web security layer

Controls:

- Email security gateway
- Anti-phishing controls
- Attachment scanning and sandboxing
- DNS/web filtering
- User awareness training

## 7. Vulnerability management layer

Controls:

- Authenticated scanning
- Risk-based prioritisation
- Patch SLAs
- Exception handling
- Remediation evidence collection

Example experience represented:

- Qualys VMDR concepts
- Nessus vulnerability scanning
- Audit issue mitigation and closure

## 8. Monitoring and detection layer

Controls:

- Centralised logging
- SIEM correlation rules
- Firewall and VPN monitoring
- Endpoint alert triage
- Use case tuning
- Incident escalation process

Example experience represented:

- Wazuh SIEM lab experience
- Central logging for network devices
- Incident response documentation

## 9. Data protection layer

Controls:

- Encryption in transit
- Encryption at rest
- Password hashing and salting
- Tokenisation or masking of sensitive fields
- Secure backup encryption
- Data retention controls

## 10. Recovery layer

Controls:

- Backup strategy
- Immutable/offline backup copies
- Recovery time objective and recovery point objective definitions
- Restore testing
- Disaster recovery runbooks
